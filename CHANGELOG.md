## 2.2.0

- Adds support for Tailwind CSS v2.2;
- Updates Tailwind CSS dependencies;
- Adds `TAILWIND_DEV_MODE` config variable;

- Updates `purge` rules in `tailwind.config.js` to include other Django app templates by default;
